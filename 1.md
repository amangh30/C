| Format Specifier | Description                      | Example Input            | Example Output                     |
| ---------------- | -------------------------------- | ------------------------ | ---------------------------------- |
| `%d`             | Signed decimal integer           | `printf("%d", 10);`      | `10`                               |
| `%i`             | Signed integer                   | `printf("%i", -5);`      | `-5`                               |
| `%u`             | Unsigned decimal integer         | `printf("%u", 100);`     | `100`                              |
| `%f`             | Floating-point number            | `printf("%f", 3.14);`    | `3.140000`                         |
| `%lf`            | Double precision floating-point  | `printf("%lf", 3.14);`   | `3.140000`                         |
| `%c`             | Character                        | `printf("%c", 'A');`     | `A`                                |
| `%s`             | String                           | `printf("%s", "Hello");` | `Hello`                            |
| `%x`             | Unsigned hexadecimal (lowercase) | `printf("%x", 255);`     | `ff`                               |
| `%X`             | Unsigned hexadecimal (uppercase) | `printf("%X", 255);`     | `FF`                               |
| `%p`             | Pointer address                  | `printf("%p", ptr);`     | `0x7ffeeaee9b60` (example address) |
| `%o`             | Unsigned octal                   | `printf("%o", 64);`      | `100`                              |
| `%e`             | Scientific notation (lowercase)  | `printf("%e", 1.23);`    | `1.230000e+00`                     |
| `%E`             | Scientific notation (uppercase)  | `printf("%E", 1.23);`    | `1.230000E+00`                     |
| `%g`             | Automatic scientific/decimal     | `printf("%g", 1.23);`    | `1.23`                             |
| `%%`             | Literal percent sign             | `printf("%%");`          | `%`                                |
