# String Functions

### 1. **`strlen()`**

* **Purpose**: Returns the length of a string (excluding the null terminator).

```c
#include <string.h>
char str[] = "Hello";
printf("%zu\n", strlen(str));  // Output: 5
```

### 2. **`strcpy()`**

* **Purpose**: Copies one string to another.

```c
#include <string.h>
char src[] = "Hello";
char dest[10];
strcpy(dest, src);
printf("%s\n", dest);  // Output: Hello
```

### 3. **`strncpy()`**

* **Purpose**: Copies up to n characters from one string to another.

```c
#include <string.h>
char src[] = "Hello";
char dest[10];
strncpy(dest, src, 3);
dest[3] = '\0';  // Null-terminate
printf("%s\n", dest);  // Output: Hel
```

### 4. **`strcat()`**

* **Purpose**: Concatenates two strings.

```c
#include <string.h>
char str1[20] = "Hello";
char str2[] = " World";
strcat(str1, str2);
printf("%s\n", str1);  // Output: Hello World
```

### 5. **`strncat()`**

* **Purpose**: Concatenates up to n characters from one string to another.

```c
#include <string.h>
char str1[20] = "Hello";
char str2[] = " World";
strncat(str1, str2, 3);
printf("%s\n", str1);  // Output: Hello Wor
```

### 6. **`strcmp()`**

* **Purpose**: Compares two strings lexicographically.

```c
#include <string.h>
int result = strcmp("apple", "banana");
printf("%d\n", result);  // Output: < 0 (apple < banana)
```

### 7. **`strncmp()`**

* **Purpose**: Compares up to n characters of two strings.

```c
#include <string.h>
int result = strncmp("apple", "apricot", 3);
printf("%d\n", result);  // Output: 0 (equal for first 3 chars)
```

### 8. **`strchr()`**

* **Purpose**: Finds the first occurrence of a character in a string.

```c
#include <string.h>
char *ptr = strchr("Hello", 'e');
printf("%s\n", ptr);  // Output: ello
```

### 9. **`strrchr()`**

* **Purpose**: Finds the last occurrence of a character in a string.

```c
#include <string.h>
char *ptr = strrchr("Hello", 'l');
printf("%s\n", ptr);  // Output: ll
```

### 10. **`strstr()`**

* **Purpose**: Finds the first occurrence of a substring in a string.

```c
#include <string.h>
char *ptr = strstr("Hello, World", "World");
printf("%s\n", ptr);  // Output: World
```

### 11. **`strtok()`**

* **Purpose**: Splits a string into tokens based on delimiters.

```c
#include <string.h>
char str[] = "Hello, World, C";
char *token = strtok(str, ", ");
while (token) {
    printf("%s\n", token);
    token = strtok(NULL, ", ");
}
// Output:
// Hello
// World
// C
```

### 12. **`sprintf()`**

* **Purpose**: Formats and stores a string into a buffer.

```c
#include <stdio.h>
char buffer[50];
sprintf(buffer, "Hello, %s!", "World");
printf("%s\n", buffer);  // Output: Hello, World!
```

### 13. **`snprintf()`**

* **Purpose**: Formats and stores a string into a buffer with size limitation.

```c
#include <stdio.h>
char buffer[10];
snprintf(buffer, sizeof(buffer), "Hello");
printf("%s\n", buffer);  // Output: Hello
```

### 14. **`memcpy()`**

* **Purpose**: Copies memory from one location to another (not specific to strings).

```c
#include <string.h>
char src[] = "Hello";
char dest[6];
memcpy(dest, src, 6);
printf("%s\n", dest);  // Output: Hello
```

### 15. **`memmove()`**

* **Purpose**: Safely copies memory (handles overlapping memory regions).

```c
#include <string.h>
char str[] = "Hello World";
memmove(str + 6, str, 5);
printf("%s\n", str);  // Output: Hello Hello
```
