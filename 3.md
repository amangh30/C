### **Array Memory Layout in C / Other Languages**

#### **1. Row-Major Order (C, C++)**

* **Definition:** Rows stored **contiguously** in memory.
* **Address formula (for `a[i][j][k]` of size `[X][Y][Z]`):**
  [
  \text{Address} = \text{base} + ((i \cdot Y \cdot Z) + (j \cdot Z) + k) \cdot \text{size_of_element}
  ]
* **Trick:** Start **left-to-right**: multiply each index by the product of dimensions to its right.
* **Example:** `a[2][3][4]` → index = (2*Y*Z + 3*Z + 4)

#### **2. Column-Major Order (Fortran, MATLAB)**

* **Definition:** Columns stored **contiguously** in memory.
* **Address formula (for `a[i][j][k]` of size `[X][Y][Z]`):**
  [
  \text{Address} = \text{base} + ((k \cdot X \cdot Y) + (j \cdot X) + i) \cdot \text{size_of_element}
  ]
* **Trick:** Start **right-to-left**: multiply each index by the product of dimensions to its left.

**Key:**

* Row-major → leftmost index dominates → C-style.
* Column-major → rightmost index dominates → Fortran/MATLAB-style.
